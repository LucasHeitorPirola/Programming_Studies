HTML

O que é?

O HTML significa Hyper Text Markup Language, isto é, uma linguagem de marcação comumente conhecida por ser utilizada no desenvolvimento front-end de sites. É importante ressaltar que ela NÃO é uma linguagem de programação, visto que ela não tem a capacidade de criar funcionalidades dinâmicas.

Qual a sua função?

Auxiliar na criação, organização e formatação da estrutura de uma página da internet.

Exemplos de tag:

  • DOCTYPE html - Tag indicadora de que o arquivo é do tipo HTML e utiliza essa linguagem.
  • html - Tag na qual podemos definir o idioma que usaremos para escrever na página e início de toda a estrutura html.
  • head - Tag que definio o cabeçalho do documento com informações que não serão exibidas dentro do conteúdo da página.
  • meta - tag de metadados que auxilia os mecanismos de pesquisa a encontrar um site na web.
  • link - tag que se usa para associar um arquivo de programação à página web.
  • title - tag que define o título da página web (nome da aba).
  • body - tag dentro da qual se construirá toda a estrutura da página web e onde será inserido o conteúdo.
  • header - tag que define o cabeçalho com informações que serão exibidas dentro do conteúdo da página.
  • div - tag que cria uma separação no corpo do documento/estrutura da página.
  • section - tag que cria uma seção no corpo da página.
  • nav - tag que cria uma barra de navegação.
  • a - tag "âncora" que se usa para associar um endereço da internet a um conteúdo.
  • p - tag que inicia um parágrafo.
  • h1, h2, h3, h4, h5, h6 - tag que inicia um título dentro do conteúdo da página de acordo com um tamanho.
  • ul - tag na qual se insere uma lista não ordenada (com pontos, quadrados, setas, por exemplo).
  • ol - tag na qual se insere uma lista ordenada (com numeração, por exemplo).
  • li - tag que cria um item dentro de uma lista.
